Shill Bot Instructions#

made by @sullyjs on Telegram#

  • First, go here and login using your Telegram phone number.
  • Once you're there, you can fill this out with any information you want. Then click "Create application".
  • You will now get your API ID and API hash, which will look like this:
  • You can now go to the folder where the shiller bot is and open "config.json".
  • Here, you need to replace the api id and api hash with your own.
  • After that, you can open the shill_message.txt file and put your shilling message.
  • Finally, run the void.exe file and fill out your phone number and give the program the code that you'll receive. You can now leave this running and it will automatically shill for you. To add additional groups, you can edit the groups.txt file, but remember that you must be in the group for the bot to work.
